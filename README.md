# openRuyi Logo

This repository provides logo assets for the openRuyi project.

## Assets

- The Git repository contains the source logo in SVG format.
- Release artifacts provide PNG exports in multiple sizes, from `16x16` to
  `1024x1024`.

The PNG files are intended to be generated from the SVG source. If there is any
difference between the SVG file and a generated PNG file, the SVG file should be
treated as the source asset.
